rasp-home-backend
=================

Python backend

# Requirements: #

* python (>= 3.2)
* sqlalchemy (>= 0.8)
  `pip-3.2 install --upgrade sqlalchemy`
* cherrypy (>= 3.2.0)
  `pip-3.2 install --upgrade cherrypy`
* passlib (>= 1.6.1)
  `pip-3.2 install --upgrade passlib`
 * netifaces
  `git clone https://github.com/kelleyk/py3k-netifaces.git
   cd py3k-netifaces
   sudo python3 setup.py install`
